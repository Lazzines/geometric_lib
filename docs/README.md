# Для файла circle.py:
- Название проекта:
 		Калькулятор геометрических параметров круга.

- Основной функционал:
 		Вычисление площади круга по заданному радиусу. Вычисление периметра круга (длины окружности) по заданному радиусу.

- Математические основы:
  - Площадь круга вычисляется по формуле: S = π × r²
  -  Периметр круга (длина окружности) вычисляется по формуле: P = 2 × π × r
  - Используется точное значение константы π из модуля math.

### Функция area(r)
- Назначение:
 			Вычисляет площадь круга по заданному радиусу.

- Параметры:
 			r (float) - радиус круга. Должен быть положительным числом.

-  Возвращаемое значение:
 			float - площадь круга, вычисленная по формуле π × r².

- Примеры вызова:
  -  Вычисление площади круга с радиусом 5
  -  result = area(5)
  - print(result)  - Вывод:78.53981633974483
 	
### Функция perimeter(r)
- Назначение:
 			Вычисляет периметр круга (длину окружности) по заданному радиусу

- Параметры:
 			r (float) - радиус круга. Должен быть положительным числом

- Возвращаемое значение:
 			float - периметр круга (длина окружности), вычисленная по формуле 2 × π × r

- Пример вызова:
  -  Вычисление периметра круга с радиусом 5
  - result = perimeter(5)
  -  print(result) - Вывод: 31.41592653589793

# Для файла rectangle.py:
- Название проекта:
 		Калькулятор геометрических параметров прямоугольника

- Основной функционал:
 		Вычисление площади прямоугольника по заданным длине и ширине. Вычисление периметра прямоугольника по заданным длине и ширине.

- Математические основы:
 		Площадь прямоугольника вычисляется по формуле: S = a × b
 		Периметр прямоугольника вычисляется по формуле: P = 2 × (a + b)
 		Где a - длина прямоугольника, b - ширина прямоугольника

### Функция area(r)
- Назначение:
 			Вычисляет площадь прямоугольника по заданным длине и ширине.

- Параметры:
    - a (float) - длина прямоугольника.  
    - b (float) - ширина прямоугольника.

-  Возвращаемое значение:
 			float - площадь прямоугольника, вычисленная по формуле a × b

- Примеры вызова:
  - Вычисление площади прямоугольника 5x3
  - result = area(5, 3)
  - print(result) - Вывод: 15
 	
### Функция perimeter(r)
- Назначение:
 			Вычисляет периметр прямоугольника по заданным длине и ширине

- Параметры:
  - a (float) - длина прямоугольника
  - b (float) - ширина прямоугольника

- Возвращаемое значение:
 			float - периметр прямоугольника, вычисленный по формуле 2 × (a + b)

- Пример вызова:
  - Вычисление площади прямоугольника 5x3
  - result = area(5, 3)
  - print(result) - Вывод: 15

  # Для файла triangle.py:
- Название проекта:
 		Калькулятор геометрических параметров треугольника.

- Основной функционал:
 		Вычисление площади треугольника по трём заданным сторонам с использованием формулы Герона.
Вычисление периметра треугольника по трём сторонам.

- Математические основы:
  - Полупериметр треугольника вычисляется по формуле:
p = (a + b + c) / 2
  - Площадь треугольника вычисляется по формуле Герона:
S = √(p × (p - a) × (p - b) × (p - c))
  - Периметр треугольника вычисляется по формуле:
P = a + b + c
  - Где a, b, c — длины сторон треугольника.

### Функция area(a, b, c)
- Назначение:
 			Вычисляет площадь треугольника по трём сторонам с использованием формулы Герона.

- Параметры:
    - a (float) — первая сторона треугольника
    - b (float) — вторая сторона треугольника
    - c (float) — третья сторона треугольника

-  Возвращаемое значение:
 			float — площадь треугольника, вычисленная по формуле Герона.

- Примеры вызова:
  - Вычисление площади треугольника со сторонами 3, 4 и 5
  - result = area(3, 4, 5)
  - print(result) # Вывод: 6.0
 	
### Функция perimeter(r)
- Назначение:
 			Вычисляет периметр треугольника по трём сторонам.

- Параметры: 
  - a (float) — первая сторона треугольника
  - b (float) — вторая сторона треугольника
  - c (float) — третья сторона треугольника


- Возвращаемое значение:
 			float — периметр треугольника, вычисленный по формуле a + b + c

- Пример вызова:
  - Вычисление периметра треугольника со сторонами 3, 4 и 5
  - result = perimeter(3, 4, 5)
  - print(result) # Вывод: 12
